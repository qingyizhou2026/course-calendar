# 📚 课程日历

一个简洁美观的课程日历管理工具，用于记录和统计课程信息。

## 功能特性

- 📅 月历视图展示所有课程
- ➕ 快速添加课程记录
- 📊 自动统计每门课程的上课次数
- ✏️ 支持编辑和删除课程记录
- 👥 支持多个学生（悠悠、露露）
- 💾 数据本地存储，无需后端

## 使用方法

直接访问 GitHub Pages 部署的网页即可使用，或下载 `index.html` 文件在本地浏览器中打开。

## 技术栈

- 纯 HTML + CSS + JavaScript
- 使用 localStorage 进行数据持久化
- 响应式设计，支持移动端
